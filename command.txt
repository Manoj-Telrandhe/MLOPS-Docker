Build docker image: docker build -t mlops-dokcer-demo .     #{-t flag tags your image with a name. ('mlops-dokcer-demo' name of file in this case), '.' lets Docker know where it can find the Dockerfile}

Run the container: docker run -p 5000:5000 mlops-docker-demo    #{-p means port 5000(what we use in app.py) : 5000(local host, can be any 5050)}

Tag your image: docker tag mlops-docker-demo mjt/mlops-docker-demo:latest     #{mjt is the uesr_name from docker hub, instead 'latest' we can give any name}

Push image to dockerhub: docker push mjt/mlops-docker-demo:latest 

Pull image from dockerhub: docker pull mjt/mlops-docker-demo:latest

Run the Pulled image: docker run -p 5000:5000 mjt/mlops-docker-demo:latest
